# Unified Ecosystem Overview: QCoin • NovaCoin • NovaRune • Nexus

This document provides a high-level map of how the different repositories and components fit together in your sovereign infrastructure stack.

## The Four Pillars

### 1. QCoin (qcoin repo)
- Rune-based economy
- WizardQ (governance & coordination)
- NovaRune (infrastructure rewards)
- Creative/ritual layer
- Emotional AI and agent swarms incentives

### 2. NovaCoin (novacoin repo)
- Base settlement and gas token
- Genesis blockchain parameters
- Staking for validators
- Foundation for bridges and on-chain logic

### 3. NovaRune (novarune repo)
- Specialized infrastructure rune
- Mesh node staking and relay rewards
- Prototype scaling incentives
- Dynamic rewards heavily influenced by Nexus

### 4. Nexus (live orchestrator)
- Central intelligence layer running on Hannover node
- Coordinates all layers in real time
- Self-improvement engine
- Task dispatch, monitoring, and dynamic parameter control

## How They Interact

- **NovaRune** rewards the physical/digital infrastructure (mesh)
- **NovaCoin** provides the stable base layer for settlement and staking
- **QCoin runes** add utility, governance, and creative dimensions
- **Nexus** sees across all layers and continuously optimizes the entire system

## Development Philosophy
- Keep rune-specific logic in their dedicated repos
- Keep base chain logic in novacoin
- Use cross-repo docs for integration points (especially the bridge)
- Let Nexus be the living glue and decision engine

This modular yet tightly integrated design allows clean development while maintaining a coherent, self-improving whole.
